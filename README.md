# Analytic Service Mock | S7 Airlines Analytics

> **Сервис автоматизированной генерации SQL-запросов для корпоративной бизнес-аналитики на базе больших языковых моделей.**

Этот репозиторий содержит реализацию Backend-части сервиса (Mock-версия для демонстрации), разработанную в рамках курсового проекта студента направления 02.03.03 «Математическое обеспечение и администрирование информационных систем» Кубанского государственного университета.

## 📋 О проекте

Сервис решает проблему высокой трудоемкости ручного написания SQL-запросов бизнес-аналитиками. Он позволяет получать аналитические выборки из корпоративной базы данных (PostgreSQL), формулируя запросы на естественном языке.

### Ключевые особенности:
*   **Text-to-SQL:** Автоматическая трансляция естественного языка в исполняемый SQL.
*   **Безопасность:** Строгая фильтрация запросов (разрешён только `SELECT`, блокировка `INSERT/UPDATE/DELETE/DROP`).
*   **Clean Architecture:** Модульная структура с разделением на Domain, Service и Infrastructure слои.
*   **Enterprise Ready:** Поддержка CI/CD, Docker, Kubernetes и структурированного логирования.

## 🛠 Технологический стек

*   **Language:** Python 3.11+
*   **Framework:** FastAPI + Uvicorn (ASGI)
*   **Database:** PostgreSQL 16 (через SQLAlchemy Async)
*   **LLM Integration:** Agno Framework + LiteLLM
*   **Validation:** Pydantic v2
*   **Testing:** pytest, unittest.mock
*   **DevOps:** Docker, Kubernetes, Helm, GitLab CI/CD
